元组tuple=( )
               不可修改  只可查询 查询操作同列表   适合存储不变的数据       元组中只有一个元素要加 ,
字典 dict
                dict ={key : value}   key键不可重复  value值可重复 且可为任意数据类型 特点：通过键快速查找值不需要遍历 比列表高效  适合存储关联数据
                常见操作
                      1.查找 方式一 字典名[键] 若键名不存在则报错
                           方式二  字典名. get(键 ,默认值  ) 若键名不存在返回默认值（可省略，默认为None）
2.修改/添加 字典名[键]=值    键名存在则修改 没有则添加
                      3.删除 方式一 字典名.pop[键名,默认值] 若键存在返回值为对应的值 若不存在返回默认值 无默认值会报错
                           方式二 字典名.clear( ) 清除字典所有元素
                      4.其他操作 字典名.keys( ) 调用后会返回所有的键  字典名.values( ) 返回所有的值  字典名.items( )返回所有的键值对（以元组的形式）
集合 set
                变量名={元素1 ,元素2 ,...}    data={ } 为空字典  data=set( )为空集合   特点：无序（没有索引 ），唯一   只能存不可变类型数据（数字，字符串，元组）， 不能存储字典，列表等可变类型
                常规操作
                      1.添加元素 add(元素) 添加单个元素
                                                      update (可迭代对象 列表 元组等) 添加多个元素 先拆分再添加
                      2.删除元素 集合名.remove(元素) 删除指定元素 不存在则会报错
集合名.discard (元素) 删除指定元素 不存在不会报错
                                         集合名.clear(元素) 清空集合所有元素
                      3.集合运算交集 a & b & c       a .intersection(b).intersection(c)
                                并集 a | b  | c       a. union(b).union(c)
                 实用技巧：去重优先使用集合(例如通过类型转换去除列表中重复的元素）以及数据对比优先选择集合
一些数据的通用操作：in/not in  len( )  del( )   字典用in判断的是键 若要判断值需.values()
                                          sum()   max()  min()只对数值类型有效 字符串按照ASCII码排序
类型转换：int() float() bool() str() eval()一般作用于去除字符串的引号 list( )  tuple() dict ()转化需要元组元素  set()自动去重
